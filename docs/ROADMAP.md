# Roadmap

## Near-term
- Add R-first training recipe for Kaggle kernels with simple MLP baseline.
- Add robust checkpoint naming conventions and metadata.
- Add reproducibility seed controls and run config snapshots.

## Mid-term
- Add CNN block support in MyTorch for stronger MNIST performance.
- Add gradient checking utilities for every trainable module.
- Add benchmark dashboard for model size vs speed vs accuracy.

## Long-term
- Build plugin-style module registry (layers, losses, optimizers).
- Add interoperability exports (ONNX, safetensors where possible).
- Add distributed training experiments and multi-dataset support.
