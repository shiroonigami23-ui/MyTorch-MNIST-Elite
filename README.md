# MyTorch: Professional Deep Learning Framework from Scratch

Welcome to **MyTorch**, a modular, high-performance deep learning library built entirely in NumPy. Designed for clarity and efficiency, MyTorch replicates the core functionality of PyTorch while maintaining extreme transparency.

## 🚀 Key Features
- **Modular Design:** Sequential containers and base Module classes for custom layers.
- **Advanced Optimization:** AdamW optimizer with L2 Weight Decay and Label Smoothing.
- **Stability & Speed:** Batch Normalization and Kaiming Initialization built-in.
- **Experiment Tracking:** Native integration with **Weights & Biases** and **Hugging Face**.

## 📊 Latest Performance
- **Dataset:** MNIST
- **Accuracy:** 98.59% (Diamond Architecture)
- **Model Registry:** [View on Hugging Face](https://huggingface.co/ShiroOnigami23/MyTorch-MNIST)
